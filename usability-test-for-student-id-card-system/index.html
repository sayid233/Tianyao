Usability test for student I.D. card system

A school project of doing usability test for student I.D. card system 

 

Background

The student I.D. card system is an important part of school management. Current student I.D. has more than 30 major functions, which covers all aspects of school life, but of course the student I.D. system is still in the process of continuous development. Most schools are still using a machine to achieve a lot of functions such as card recharging, paying air conditioning fee, paying Internet fee…

We found that the usability and ease of use of the machine in SCUT are pretty bad, so we are gonna try to improve the system through the usability test.

Old structure

We tried all the functions of the machine and concluded the information structure of the student I.D. system now.

Objectives

  • Evaluate the ease of use of the basic functional operation procedures such as air-conditioning fee recharge and Internet fee payment in the student I.D. card machine through the usability test.
  • Obtain user feedback through user interviews and user satisfaction evaluation form.
  • Analyze the test results, propose improvement suggestions to the process of air-conditioning fee recharge, Internet fee payment, and improve the interaction design and user interface design.
